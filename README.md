Password Generator GUI

This is a simple desktop application built using Python and Tkinter that helps users generate strong and secure passwords quickly. The project was created to practice Python programming, GUI development, and basic cybersecurity concepts.

Features

- Generate random and secure passwords
- Choose the desired password length
- View password strength (Weak, Medium, or Strong)
- Copy generated passwords to the clipboard with one click
- Easy-to-use graphical interface

Technologies Used

- Python
- Tkinter

How to Run

1. Make sure Python is installed on your system.
2. Download or clone this repository.
3. Open the project folder in VS Code or a terminal.
4. Run the following command:

python password_gui.py
